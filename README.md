This is a static website built with vanilla HTML, CSS, and JavaScript for optimal performance and easy deployment.

### Prerequisites

- A web browser (Chrome, Firefox, Safari, Edge)
- Optional: A local web server for testing

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/REDPHANTOMOPS/RedPhantomOps.git
cd RedPhantomOps
```

2. Open the website:
   - **Option 1:** Simply open `index.html` in your web browser
   - **Option 2:** Use a local web server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

3. Navigate to `http://localhost:8000` (if using a server)
